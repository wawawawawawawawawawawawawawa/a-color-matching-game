# a-color-matching-game

A fun and educational color memory game built with Pygame. Test how well you can remember and match colors!

The game is a simple application that challenges players to recreate the color that is shown to them in a brief amount of time. Players will need to use their understanding of color to change the intensity and balance of the three primary colors (red, green, blue) to recreate the color that they saw. The closer their recreated color is to the original, the higher their score, with a perfect match earning 100%.
The game is fun for casual play and also for color training, which is important for photographers, photo editors, painters, etc. who wants to train their sense of color accuracy - a great warm-up exercise for creative professionals!

How to play:
1. **Show Phase**: Remember the displayed target color for 5 seconds.
2. **Adjust Phase**: Use the RGB sliders to match the color you remember.
3. **Result Phase**: See your accuracy for that round.
4. Play **3 rounds** and get your final total score with a fun verdict!
